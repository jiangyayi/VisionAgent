# 視障 AI 預警輔助系統 (VisionAgent System)

### 🚀 東海大學深度學習應用課程專題 (Course Project)
[![Python](https://img.shields.io/badge/Python-3.9+-blue.svg)](https://www.python.org/)
[![YOLO](https://img.shields.io/badge/Model-YOLO11n-green.svg)](https://github.com/ultralytics/ultralytics)
[![API](https://img.shields.io/badge/Cloud%20API-iTHU%20Vibe%20120B-orange.svg)]()

本專案旨在解決傳統視障輔具（如白手杖）的物理探測極限，以及現存雲端 AI 方案因網路傳輸導致的預警延遲痛點。系統採用 **邊雲協作（Edge-Cloud Collaboration）** 架構，兼顧本地端的即時避險速度與雲端的深度語義理解。

---

## 📌 專案精簡摘要 (TL;DR)
* **基於 Python 的低延遲避險架構**：開發端對端視障全場域輔助系統。
* **智慧單目測距與幾何校正**：實作 YOLO11n 雙模型並行偵測，結合相似三角形測距與 Horizon 防低頭補償演算法。
* **非同步多進程語音管理**：設計動態語音分級與 Queue 隊列「暴力清洗」機制，從根本解決語音播報造成的影像推論延遲。
* **邊雲協作多模態語義理解**：整合 iTHU 平台雲端 Vibe 大模型，提供紅綠燈狀態判定與複雜路況文字之深度解讀。

---

## 🛠️ 核心技術與開發亮點

### 1. YOLO11n 雙模型並行策略 (本地推論)
系統在本地端同時調用兩組 YOLO 模型以兼顧算力與精度：
* **基礎層 (COCO 預訓練權重)**：穩健捕捉行人、車輛及家具等 80 類通用環境目標。
* **專項層 (自定義 `best.pt` 權重)**：專注於階梯、坑洞、路樁、交通錐等 6 類對視障者具高度風險的地形與動態障礙物，有效避免單一模型因類別過多導致的辨識精度下降。

### 2. 智慧單目測距與 Horizon 防低頭校正
* **單目空間建模**：不依賴高耗能的深度感測器，僅透過單目相機結合相似三角形原理推算物體的真實物理距離。
* **防低頭補償**：針對穿戴式相機易因人體擺動而向下傾斜的實務痛點，引入地平線像素補償機制（`CAMERA_TILT_OFFSET`），大幅修正因低頭造成的測距失真。

### 3. 非同步多進程架構與語音隊列清洗
* **運算任務解耦**：為了解決語音播報（TTS）造成影像偵測畫面凍結與延遲的硬體瓶頸，系統將「視覺推論行程」與「語音管理行程」完全隔離運作。
* **隊列清洗機制 (Queue Flushing)**：在獨立語音行程中，當新的緊急危險發生時，系統會自動拋棄 Queue 中積壓的過時舊訊息，確保使用者聽到的永遠是當下最即時的路況預警。

### 4. iTHU 雲端 Vibe 大模型協同 (120B VLM)
* **高吞吐智慧解讀**：針對特定目標（如紅綠燈狀態）或複雜路況，系統會開啟背景獨立執行緒，透過 API 串接雲端 Vibe 多模態大模型（採用 NVFP4 量化，吞吐量達 150 tokens/s）。
* **行動指令化**：利用其強大的零樣本視覺理解能力解析路牌告示，並透過 Prompt 工程將複雜場景轉譯為 **10 字以內**、具備行動建議的極簡導引（例如：「請通行」、「前方施工請繞開」）。

---

## 📈 系統實作成果
本專案成功整合「本地即時物件偵測」、「單目幾何空間分析」與「雲端大模型解讀」。經實機影片驗證，系統能在複雜室內外場景下，對高風險地形與動態障礙物進行精確測距與方位判定，並透過高時效、零積壓的語音分級指令，顯著降低視障者的碰撞風險，切實提升其獨立行進的安全性與行動自主權。
